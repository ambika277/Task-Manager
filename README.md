# Task Manager

A simple Java console-based Task Manager application that allows users to manage their daily tasks. This project demonstrates the use of Java concepts such as classes, objects, collections, loops, and user input.

## Features

- Add new tasks
- View all tasks
- Update existing tasks
- Delete tasks
- User-friendly console interface

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Collections Framework (ArrayList)
- Scanner Class

## Project Structure

```
Task-Manager/
│── Task.java
│── TaskManager.java
│── README.md
```

## Prerequisites

- Java JDK 8 or later
- VS Code / IntelliJ IDEA / Eclipse

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/ambika277/Task-Manager.git
```

2. Navigate to the project folder:

```bash
cd Task-Manager
```

3. Compile the Java files:

```bash
javac *.java
```

4. Run the application:

```bash
java TaskManager
```
